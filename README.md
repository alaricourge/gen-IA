# gen-IA
implémentation d'un graph rag sur des articles de lois francais 
## Description
Ce projet implémente un graphe RAG (Retrieval-Augmented Generation) basé sur des articles de lois français. L'objectif est de fournir une solution permettant de générer des réponses précises et contextualisées en utilisant des données légales.

## Fonctionnalités
- Extraction et traitement des articles de lois français.
- Construction d'un graphe RAG pour améliorer la génération de réponses.
- Intégration d'un modèle de génération de langage pour fournir des réponses contextuelles.

## Prérequis
- Python 3.13
- Bibliothèques nécessaires dans le notebook

## Installation
1. Clonez le dépôt :
    ```bash
    git clone https://github.com/alaricourge/gen-IA.git
    ```
2. Accédez au répertoire du projet :
    ```bash
    cd gen-IA
    ```
